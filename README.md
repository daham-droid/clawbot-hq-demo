Live demo of ClawBot HQ - AI-powered office simulation.

## What is ClawBot HQ?

A browser-based idle game where 5 AI agents work in your virtual office:
- **Scout** - Researches trends
- **Scribe** - Creates content
- **Architect** - Builds features
- **Analyst** - Analyzes data
- **Director** - Plans strategy

Generate daily reports to paste to your AI assistant for better suggestions.

## Features

- Topic memory system
- Contextual conversations
- Daily productivity reports
- 8 office upgrades
- Leveling system
- Batch commands
- Smooth animations
